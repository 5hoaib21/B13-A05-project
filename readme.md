1️⃣ What is the difference between var, let, and const?
Ans: 
var: It has function scope. Variables declared with var can bee redeclar and reassign.

let: It works within block scope. You cannot redeclare it in the same scope, but reassignment is allowed.

const: It is also block scoped, but it can not be redeclar or reassigne after the initial value is set.

3️⃣ What is the difference between map(), filter(), and forEach()?
Ans:

map: Create and return a new array afteer trasforming each element of the orignal array.

forEach: Run a function on each element, but does not return  new array.

filter: Return a new array containing only the elements that match a specific condition.
